## Line Numbers

default config: `site.config.js` -> `config.markdown.lineNumbers`

**Input**

````mdx raw
```js lineNumbers={false}
// lineNumbers is disabled
const line2 = 'This is line 2'
const line3 = 'This is line 3'
```
````

````mdx raw
```js lineNumbers
// lineNumbers is enabled
const line2 = 'This is line 2'
const line3 = 'This is line 3'
```
````

**Output**

```js lineNumbers={false}
// lineNumbers is disabled
const line2 = 'This is line 2'
const line3 = 'This is line 3'
```

```js lineNumbers
// lineNumbers is enabled
const line2 = 'This is line 2'
const line3 = 'This is line 3'
```

## Title

**Input**

````mdx raw
```ts title="src/utils/clipboard.ts"
export function copyToClipboard(text: string) {
  if (navigator && navigator.clipboard && navigator.clipboard.writeText)
    return navigator.clipboard.writeText(text)
  return Promise.reject('The Clipboard API is not available.')
}
```
````
**Output**

```ts title="src/utils/clipboard.ts"
export function copyToClipboard(text: string) {
  if (navigator && navigator.clipboard && navigator.clipboard.writeText)
    return navigator.clipboard.writeText(text)
  return Promise.reject('The Clipboard API is not available.')
}
```

## Line Highlighting in Code Blocks

**Input**

````mdx raw
```text lineNumbers highlights='1,3-5'
line 1
line 2
line 3
line 4
line 5
line 6
line 7 // [!code highlight]
```
````

**Output**

```text lineNumbers highlights='1,3-5'
line 1
line 2
line 3
line 4
line 5
line 6
line 7 // [!code highlight]
```

## Colored Diffs in Code Blocks

**Input**

````mdx raw
```js
const data = {
  message: 'Removed', // [!code --]
  message: 'Added', // [!code ++]
}
```
````

**Output**

```js
const data = {
  message: 'Removed', // [!code --]
  message: 'Added', // [!code ++]
}
```
